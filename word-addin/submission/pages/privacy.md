## Draft Content

Refx for Word is a companion add-in for the Refx desktop application.

The add-in reads reference metadata from the Refx desktop app running on the same computer through a local companion bridge at `http://127.0.0.1:38474`.

The add-in stores citation state inside the active Word document using Word content controls and a custom XML part. This state may include stable citation group ids, Refx reference ids, reference metadata, citation settings, and linked work information.

The add-in does not require a cloud account for the current beta architecture and does not send Word document contents to a Refx cloud service.

Before public publication, replace this draft with the final legal privacy policy, including:
